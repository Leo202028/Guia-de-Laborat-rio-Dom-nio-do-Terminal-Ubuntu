# Guia-de-Laborat-rio-Dom-nio-do-Terminal-Ubuntu





🖥️ Domínio do Terminal no Ubuntu
O Terminal é uma das ferramentas mais poderosas do Ubuntu. Dominar seus comandos permite controlar o sistema com rapidez, automatizar tarefas e resolver problemas com eficiência.

🧑‍💻 1. O que é o Terminal?
O Terminal é uma interface de linha de comando (CLI – Command Line Interface) onde você interage com o sistema digitando comandos, em vez de usar o mouse.
No Ubuntu, o terminal padrão utiliza o shell Bash (Bourne Again Shell).

⚙️ 2. Comandos Básicos

Aqui estão alguns comandos essenciais para começar:


| Comando | Função                   | Exemplo            |
| ------- | ------------------------ | ------------------ |
| `ls`    | Lista arquivos e pastas  | `ls -l`            |
| `cd`    | Navega entre diretórios  | `cd Documentos`    |
| `pwd`   | Mostra o diretório atual | `pwd`              |
| `mkdir` | Cria uma pasta           | `mkdir nova_pasta` |
| `rm`    | Remove arquivos/pastas   | `rm arquivo.txt`   |
| `cp`    | Copia arquivos           | `cp a.txt b.txt`   |
| `mv`    | Move ou renomeia         | `mv a.txt pasta/`  |

No Linux, cada arquivo tem permissões:

r (read) → leitura
w (write) → escrita
x (execute) → execução

📦 4. Gerenciamento de Pacotes

O Ubuntu usa o gerenciador APT para instalar programas:


sudo apt update
sudo apt upgrade
sudo apt install nome-do-programa
